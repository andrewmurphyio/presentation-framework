# Presentation Framework - Claude Code Quick Reference

## Ralph Wiggum Commands

**Plan:**
```
/ralph-wiggum:ralph-loop "Read PROMPT_plan.md and follow its instructions exactly." --max-iterations 2
```

**Build:**
```
/ralph-wiggum:ralph-loop "Read PROMPT_build.md and follow its instructions exactly." --max-iterations 10
```

**Cancel:**
```
/ralph-wiggum:cancel-ralph
```

## Quick Commands

- `npm test` - run tests
- `npm run dev` - start dev server (http://localhost:5173)
- `npm run build` - production build
